Steps to deploying to Heroku

with automatically deploying from pushing to github

  1. Create new github-repo
  2. Push content to github
  3. Connect from Heroku to gitHub
  4. Enable automatic deployment
  5. npm install @angular/cli@latest @angular/compiler-cli --save
